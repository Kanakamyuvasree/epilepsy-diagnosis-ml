# 🧠 Machine Learning-based Epilepsy Diagnosis System

## 📌 Overview

This project presents an AI-driven system for detecting epilepsy using EEG/MRI data. It leverages Machine Learning and Deep Learning models such as Convolutional Neural Networks (CNN) and Support Vector Machines (SVM) to classify brain signals/images into epileptic and non-epileptic categories.

The system automates diagnosis, reduces manual effort, and improves accuracy using data-driven techniques.

---

## 🚀 Features

* 🔍 Automated epilepsy detection using ML/DL models
* 🧠 CNN-based feature extraction for image data
* 📊 SVM for classification
* ⚙️ Data preprocessing (normalization, filtering, augmentation)
* 📈 Performance evaluation (Accuracy, Precision, Recall, F1-score)
* 🖥️ User interface for uploading data and viewing predictions

---

## 🏗️ System Architecture

```
Input Data (EEG / MRI)
        ↓
Data Preprocessing
        ↓
Feature Extraction (CNN / Signal Processing)
        ↓
Model Training (SVM / CNN)
        ↓
Prediction (Epileptic / Non-Epileptic)
        ↓
Result Visualization (UI / Output)
```
