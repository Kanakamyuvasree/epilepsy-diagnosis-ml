# 🧠 Epilepsy Diagnosis using Machine Learning

## 📌 Problem
Manual diagnosis of epilepsy using EEG data is time-consuming and requires expert neurologists.

## 🚀 Solution
Developed an ML-based system to classify EEG signals into:
- Epileptic
- Non-Epileptic

## 🛠 Tech Stack
- Python
- TensorFlow
- Scikit-learn
- NumPy, Pandas

## 📊 Dataset
Kaggle Epilepsy EEG Dataset
